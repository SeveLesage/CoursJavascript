Placement et action du JS

Ici, le JS est placer directement dans Body puis dans les differentes balise des elements auxquels nous voulon appliquer du JS. Il va falloir apres définir l'action es le résultat que l'on veut obtenir. Ce n'est pas une methode recommander, c'est similaire a faire du css dans les l'élément directement

On a définit ici l'action d'alerte quand on clique sur le premier bouton. Une fenetre apparaît avec un message que l'on definit dans le code. lorsque l'on clicque sur le deuxiemen bouton, un paragraphe sera ajouté à l'endroit souhaité.
