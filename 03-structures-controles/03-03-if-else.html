1 - Explications

La structure conditionnelle if () {{}} else {{}} (en français, SI..SINON) sera plus compllete car on va executé un bloc de code si le test renvoie true ou un autre dans le cas contraire

Exemple

  • Dans notre premiere condition, vous allons demander au navigater : SI variable x qui contient 500 est plus grande que 1 article ce premier bloc SINON affiche ce deuxieme bloc

  • Ici on va demander au navigateur : SI la variable y qui contient la valeur 0.5 est plus grande que 20 affiche ce premier bloc de code SINON affiche le deuxieme
