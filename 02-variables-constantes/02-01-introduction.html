Introduction variables

1 - Qu'est ce qu'un variable ?

Une variable est un conteneur servant a stocké des information de maniere temporaire ça peut-être du texte (une chaine de caractere) Des nombres etc... La variable varie ! elle stocké différentes valeurs au fil du temps

Une variable et sa valeur sont deux choses différentes.

2 - Les régles de déclaration

On dit que l'on déclare une variable comme en PHP. On va déclarer une variable grâce a des mot clés.

A - Les mots clé

  • Let : exemple
    let age = 28;
  • Var : exemple
    var couleur = 'jaune';

B - Quelques régles

  • Le nom d'une variable doit toujours commencer par une lettre ou un underscrore
  • Le nom d'une variable ne doit pas contenir de caractére spéciaux
  • Il n'y a pas d'espace dans la déclaration d'un variable.

3 - Initialiser une variable

Quand on assigneune valeur a une variable vide, on dit que l'on stocke une valeur ou qu'on initialise une variable

Les différences entre Let et var

Let est plus recent que var par défaut nous utiliserons let.

A - La remontée des variables (avec var)

Avec var, nous ne somme pas obligé de déclarer la variable avant de la manipuler ou de la l'affecter

  • Ceci fonctionne :
    nom = 'Perinel';
    var nom;
  • Ceci ne fonctionne pas :
    ville = 'Paris';
    let ville;

B - La redéclaration des variables

Avec vr on avait la possibilité de déclarer une même variable en utilisant à chaque fois var, ce qui avait pour effet de modifier la valeur stockée. On ne peut pas le faire avec Let

  • Ceci fonctionne
    var chien = 'Max';
    var chien = 'medor';
  • Ceci ne fonctionne pas
    let chat = 'ObiWan';
    let chat = 'Skywalker';

C - La portée des variables

La portée d'une variable désigne l'endroit ou cette variable va pouvoir être utiliser dans un script. Nous rentrerons plus en détaille lorsque nous verrons les fonctions en javascript.

Conclusions

La syntaxe de céclaration des variables avec let est la nouvelles syntaxe var est vouée à disparaître. Vous devez donc normalement toujours utilisér le mot clé let.

5 - Utilité des variables

Il sera pratique de stocké différent type de données ou d'information dans une variables

Exemple : avec la fonction prompt(), on pourra récupéré une donnée envoyer par l'utilisateur et cette donnés ser stocké grâce à une variable.
