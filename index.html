JavaScript : un langage aux multiples usages

Introduction

  1. Un langage dynamique
  2. Un langage principalement côté client
  3. Un langage interprété

I. langage dynamique

Le JavaScript va nous permettre de manipuler des contenus HTML ou des styles CSS et de les modifier en fonction de divers événementss ou variables. Un événement peut être par exemple un clic de l'utilisateur à un certain endroit de la page et une variable peut être l'heure de la journée.

II. langage principalement côté client

La chose importante a retenir ici est que le javascript est un langage principalement utilisé côté client. cependant, il est utilisé, et de plus en plus, côté serveur pour cela il faudras mettre en place un environnement favorable à l'utilisation côté serveur (grâce à NodeJS par exemple.)

III. langage interprété

Le javascript est un langage interprété cela signifie qu'il va pouvoir etre interprété directement sous réserve que l'on posséde le logiciel interpréteur. Pas de panique ici : tous les navigateurs connus possèdent leurs interpréteurs JS.

Javascript versus Java !!! Attention !

Encore aujourd'hui, certaines personnes ont tendance à confondre les deux langages ou à utiliser "Java" pour parler de JS. Retenez ici, que ces deux langage, bien que syntaxiquement assez proches à la base, reposent sur des concepts ainsi que sur des tâches totalement différents.
